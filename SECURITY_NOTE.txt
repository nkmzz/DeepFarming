Smartfarm static site security note

Applied measures:
- HTML meta noindex/nofollow/noarchive/noimageindex
- robots.txt Disallow: /
- JavaScript password gate on every HTML page
- .nojekyll for GitHub Pages static hosting

Important limitation:
This is only a casual-access gate for GitHub Pages. It is not real server-side authentication.
Anyone with repository/source access can inspect files and bypass it. Use Cloudflare Access, private hosting, or a private repository with controlled Pages support for real access control.

Password is not stored in this file. The active hash is inside auth.js.
